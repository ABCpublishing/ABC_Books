# Google Sign-In Setup Guide for ABC Books

This guide explains how to configure the Google Cloud Console to allow Google Sign-In on your local machine and your production website.

## 1. Access Google Cloud Console
1.  Go to the [Google Cloud Console](https://console.cloud.google.com/).
2.  Ensure your project (e.g., "ABC Books") is selected in the top project dropdown.

## 2. Configure OAuth Credentials
1.  In the left sidebar, go to **APIs & Services** > **Credentials**.
2.  Under **OAuth 2.0 Client IDs**, find your existing ID and click the **Pencil (Edit)** icon.

## 3. Add Authorized Origins (CRITICAL)
This step fixes the `unregistered_origin` error you saw in the console.

### A. Authorized JavaScript origins
Scroll down to this section and add the following URLs one by one:
*   `http://localhost:3001` (For local testing)
*   `http://127.0.0.1:3001`
*   `https://abcbooks.store` (For your live website)

### B. Authorized redirect URIs
Add the same URLs here as well:
*   `http://localhost:3001`
*   `https://abcbooks.store`

**Click SAVE at the bottom.**

## 4. Update Your Code (If changing IDs)
If you ever create a **new** Client ID, you must update it in two places in your code:

1.  **Backend**: `backend/.env`
    ```env
    GOOGLE_CLIENT_ID=your_new_id_here
    ```
2.  **Frontend**: `js/auth/user-auth-api.js` (around line 248)
    ```javascript
    const GOOGLE_CLIENT_ID = 'your_new_id_here';
    ```

## 5. Deployment Note (AWS)
When you deploy to AWS:
1.  Get your AWS assigned URL (e.g., `https://abc-books-xyz.awsrunner.com`).
2.  Come back to the [Google Cloud Console](https://console.cloud.google.com/).
3.  Add that AWS URL to the **Authorized JavaScript origins** list.
4.  Wait 5 minutes for Google to update its servers.

## 🚦 Troubleshooting
*   **"Origin not allowed"**: Double-check that there is no trailing slash in the URL (e.g., use `http://localhost:3001` NOT `http://localhost:3001/`).
*   **Button not appearing**: Open your browser in **Incognito Mode**. Google sometimes caches old permission settings.
